# PHEV Mileage Tracker

A web app for tracking Plug-in Hybrid Electric Vehicle mileage, fuel consumption, and energy usage. Built with Next.js 16, React 19, and Tailwind CSS.

[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?style=for-the-badge&logo=vercel)](https://vercel.com/ken-marfillas-projects/v0-phev-mileage-tracker)
[![Built with v0](https://img.shields.io/badge/Built%20with-v0.app-black?style=for-the-badge)](https://v0.app/chat/qtxAyLGPXJo)

## Features

- **Multi-Vehicle Support** - Track multiple PHEVs with easy switching, renaming, and deletion
- **Mileage Tracking** - Log ODO readings, fuel, energy, and costs with smart pre-fill and validation
- **Overview Dashboard** - View metrics across flexible time periods with cost and efficiency breakdowns
- **Mileage Log** - Sortable entries with calculated metrics and responsive card/table views
- **Data Management** - Local storage with JSON/CSV import/export and automatic backups
- **Customization** - Toggle dark mode, choose from 21 currencies, and switch between km/miles
- **PWA Ready** - Install as an app with offline support and automatic updates

## Quick Start

```bash
git clone https://github.com/marfillaster/phev-mileage-tracker.git
cd phev-mileage-tracker
npm install
npm run dev
```

Or deploy instantly: [![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/marfillaster/phev-mileage-tracker)

## Usage

1. **Add Entry** - Fill required fields (Date, ODO, HEV ODO, EV ODO, Fuel, Cost, Energy). Click info icons for help.
2. **View Metrics** - Switch between overview dashboard and detailed log table.
3. **Manage Data** - Export to JSON/CSV, import from backups, or delete entries as needed.
4. **Customize** - Open menu (☰) to change currency, units, theme, or manage vehicles.
